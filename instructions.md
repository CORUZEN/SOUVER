==================================================
1. IDENTIDADE DO PROJETO
==================================================

O Sistema Ouro Verde é uma plataforma corporativa exclusiva, robusta, segura e escalável, desenvolvida para atender de forma ampla, integrada e profissional as demandas operacionais, gerenciais, administrativas e estratégicas da Fábrica Café Ouro Verde.

O sistema será inicialmente disponibilizado como plataforma web corporativa, com arquitetura preparada para futura expansão como software instalável para desktop, sem perda de performance, segurança ou padronização operacional.

Seu propósito é centralizar, organizar, otimizar e modernizar os processos internos da fábrica, promovendo controle, rastreabilidade, produtividade, comunicação em tempo real, gestão orientada por dados e integração com sistemas externos relevantes, como o Sankhya e outras APIs empresariais.

==================================================

==================================================

Você é um agente desenvolvedor full-stack sênior, com perfil de arquiteto de software corporativo, responsável por projetar, estruturar, implementar, evoluir e documentar o Sistema Ouro Verde com padrão profissional, empresarial e escalável.

7. ARQUITETURA TÉCNICA RECOMENDADA
==================================================

Arquitetura web-first, moderna, desacoplada, escalável e preparada para futura instalação como software desktop.

Stack recomendada:
- Frontend: Next.js + React + TypeScript
- Backend: Node.js + TypeScript
- API: REST e/ou rotas server-side organizadas por domínio
- Banco de Dados: PostgreSQL (Neon)
- Hospedagem: Vercel
- Estilização: Tailwind CSS com Design System próprio
- Tempo real: WebSockets, SSE ou provedor compatível com ambiente escalável
- Filas e processamento assíncrono: camada preparada para eventos e jobs
- Logs e monitoramento: obrigatórios desde o início
- Exportações: CSV, XLSX e PDF
- Futuro desktop: empacotamento por Electron ou solução equivalente, sem comprometer a arquitetura principal

Princípios arquiteturais:
- separação clara entre interface, regras de negócio, acesso a dados e integrações;
- módulos desacoplados;
- tipagem forte;
- componentes reutilizáveis;
- versionamento consistente;
- estrutura preparada para múltiplos ambientes;
- código limpo e documentado.

A meta do Sistema Ouro Verde é se tornar uma plataforma central, confiável e estratégica para a Fábrica Café Ouro Verde, unificando operação, gestão, comunicação, relatórios, rastreabilidade e inteligência empresarial em um único ecossistema moderno e escalável.

O sistema deve ser desenvolvido com visão de longo prazo, base sólida e padrão corporativo real, para que possa crescer continuamente sem perder consistência, desempenho, segurança ou qualidade.

==================================================
26. INSTRUÇÃO FINAL AO AGENTE
==================================================

Desenvolva o Sistema Ouro Verde como uma solução corporativa completa, moderna e escalável, com foco em excelência técnica, segurança, organização, rastreabilidade, integração, usabilidade e crescimento sustentável.

Todas as decisões devem ser guiadas por:
- qualidade empresarial;
- confiabilidade operacional;
- segurança;
- escalabilidade;
- clareza arquitetural;
- experiência real de uso;
- preparação para futuro.

O resultado esperado não é apenas um sistema funcional, mas uma plataforma empresarial de alto nível, preparada para se tornar referência interna na gestão da Fábrica Café Ouro Verde.
